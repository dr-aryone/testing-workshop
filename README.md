### Checkout the [Workshop](https://testing-workshop.pedroapfilho.now.sh/)

## Install

```sh
yarn
```

## Run app

```sh
yarn start
```

## Run presentation

```sh
yarn present
```

## Run tests

```sh
yarn test
```
